# AI Travel Buddy

**Multi-Agent Travel Planner Demo**  
Automated trip planning: flights, hotels, itinerary, packing lists, and reminders.

## Setup

```bash
git clone <repo_url>
cd ai-travel-buddy
python -m venv venv
source venv/Scripts/activate
pip install -r requirements.txt
```
